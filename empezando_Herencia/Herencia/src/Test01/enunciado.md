Tienes esta clase ya implementada:
javapublic class Persona {
private String nombre;
private int edad;
private int peso;
}
Crea una clase Estudiante que herede de Persona y añada los siguientes atributos:

curso (String)
nota (double)

La clase debe ser visible desde cualquier parte del programa, y los atributos deben estar encapsulados (privados).
